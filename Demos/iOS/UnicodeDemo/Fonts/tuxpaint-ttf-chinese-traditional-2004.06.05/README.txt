Chinese (Traditional) TrueType Font (TTF) for Tux Paint

Bill Kendrick
bill@newbreedsoftware.com
http://www.newbreedsoftware.com/tuxpaint/

June 5, 2004 - June 5, 2004


This font is required to run Tux Paint in Chinese (Traditional).
(e.g., with the "--lang traditional-chinese" option)

To install, run "make install" as the superuser ('root').
The font file will be placed in the /usr/share/tuxpaint/fonts/locale/
directory.
